# WeatherScope Pro

WeatherScope Pro is a modern weather analytics dashboard that provides live weather information, air quality status, temperature trends, weather condition charts, and smart weather insights.

## Live Site

Live Demo: [Weather Analytics Dashboard](https://weather-analytics-dashboard12.netlify.app/)

## Project Overview

WeatherScope Pro is a frontend-based weather dashboard built using HTML, CSS, and JavaScript. It uses the OpenWeather API to fetch real-time weather, forecast, and air quality data. The dashboard includes animated weather backgrounds, interactive charts, login authentication, and a clean responsive UI.

## Features

- User login page with demo credentials
- Weather dashboard with live city search
- Current temperature, humidity, wind speed, and AQI
- Temperature trend line chart
- Upcoming temperature bar chart
- Weather condition doughnut chart
- Humidity and wind trend chart
- Dynamic weather-based background animation
- AI-style weather insight messages
- Responsive design for desktop and mobile
- Toast notifications for login and weather errors
